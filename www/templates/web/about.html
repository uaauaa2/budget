О сайте

Это еще один сайт для управления семейным бюджетом

Особенности/Ограничения:

  • база данных бюджета представляет собой json объект
  • база данных бюджета хранится в вашем личном хранилище Яндекса для приложений, соответственно у Вас должен быть Яндекс аккаунт и вы должны разрешить сайту хранить там данные
  • вся логика обработки находится на клиенте, все вычисления и работа с базой происходят в браузере
  • Также есть простой клиент для андроида, использующий абсолютно те же скрипты, что и веб-версия, который позволяет добавлять в базу данных информацию о расходах.
  • 4 уровня иерархии статей расходов, верхний уровень иерархии может содержать только один элемент
  • Модули:
    • Настройки. Служит для авторизация приложения в Яндексе, выбора активной базы данных, просмотра журнала обработки данных
    • Редактирование статей расходов
    • Расходы
    • Доходы
    • План расходов по месяцам
    • Баланс
    • План
